# Lab 5 - Series Tracker

Aplicación web hecha en Go para llevar control de episodios de series usando Server Side Rendering.

## Funcionalidades

- Listar series
- Agregar nuevas series
- Incrementar episodios (+1)
- Base de datos SQLite
- Templates HTML

## Ejecutar el proyecto

Clonar el repositorio:
EN MI CASO
git clone https://github.com/Deniss3232/lab5-web-21151.git
cd lab5-web-21151
go run cmd/server/main.go
