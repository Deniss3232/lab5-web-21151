# Lab 5 - Series Tracker

Aplicación web hecha en Go para llevar control de episodios de series usando Server Side Rendering.

## Funcionalidades

- Listar series
- Agregar nuevas series
- Incrementar episodios (+1)
- Base de datos SQLite
- Templates HTML
